# JF Cartuchos e Recargas — Início

Objetivo do projeto foi recriar a home de um site real, priorizando por inicio um layout estático, tipografia, cores, espaçamentos e hierarquia visual. O site foi desenvolvido apenas com HTML e CSS (mobile-first), sem uso de JavaScript ou frameworks. Nosso site de referência foi o {https://ligmais.net/jf-cartuchos-e-informatica-samambaia-norte}
